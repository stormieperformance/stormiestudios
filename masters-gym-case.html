Masters Gym

Kickboxing & Muay Thai club · Kungsholmen, Stockholm

Where this stands: the technical build below is live and working today, real CMS, real code, real client meetings. The gym's own photography and hero video are still being finalized on their end, so the visual polish on the live site isn't final yet. Publishing this now on purpose, so what you're reading is what was actually built, not a finished showreel.

The challenge

A dated site with no clear path from visitor to member, nothing for corporate inquiries, and content that depended on a developer to update.

The approach

A fully custom, hand-coded, bilingual build structured around conversion, with a content system matched to how the club actually works day to day.

i.

Built for conversion

Separate paths for members, families, and corporate training, each with its own call to action, rather than one generic contact form.

ii.

Self-service news

Moved onto a real headless CMS with photo support. Published and confirmed live end to end, Studio to site, no developer required.

iii.

Schedule that fits how they work

Kept on a spreadsheet on purpose, a deliberate call, not a shortcut. It's genuinely the fastest way to edit fast-moving weekly class times.

iv.

Social-first design

A live Instagram feed built in, meeting members where they already look first, which matters a lot in the fitness industry.

Outcome so far
  • Clear conversion paths for individual members and business customers alike
  • News fully self-service, confirmed working with a real published post
  • No placeholder or unfinished content anywhere on the site

How it's actually built

  • Hand-coded HTML / CSS / JS, no page builder
  • Headless CMS via Sanity, queried with GROQ
  • Schedule sync via Google Sheets, deliberately kept separate from the CMS
  • Bilingual EN / SV throughout, one i18n system
  • Membership signups via GymControl
  • Contact handled through Formspree
  • Deployed on GitHub Pages, custom domain in progress
  • Light / dark mode, fully responsive, mobile-first
