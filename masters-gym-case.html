Built

One club.
One system.

The old site ran on Squarespace, so the club could technically edit it themselves. The real problem wasn't developer-dependency, it was no bilingual support, no clear path from visitor to member, and content that went stale anyway.

The rebuild is fully custom and hand-coded, structured around conversion, with a content system matched to how the club actually works day to day, not a generic template with a logo swapped in.

News now runs on a real CMS with photo support, confirmed live end to end. The class schedule stays on a spreadsheet, the fastest way to edit fast-moving weekly class times, with checks in place so a typo can't break the page.

Live build, real CMS, real code, real client meetings.
