One gym.
One system.

Most Muay Thai gyms run on scattered tools a basic website, a separate booking platform, memberships managed through email, and a shop that lives somewhere else entirely. Nothing feels connected.

This concept brings it together. One site with a hero video that sets the tone, a schedule members can actually use, membership plans with clear paths to conversion, a branded shop and a news section that keeps the club feeling alive.

The Instagram feed updates automatically through Behold.so so as long as the gym posts on Instagram, the website stays fresh without anyone touching the backend.

Concept project all parts built and live at smt-demo.

How it works

Six parts. One experience.

01

Brand layer

Website

Hero video that sets the tone before a word is read. Clean navigation, about section, coaches and contact all in one scroll.

02

Class access

Schedule

Today and tomorrow tabs. Each class card shows time, name and level with a direct booking entry point. No external link, no friction.

03

Member flow

Memberships

Six plans covering every type of member drop-in, unlimited, beginner, student, youth and pack. Clear pricing, clear path to start.

04

Commerce layer

Shop

Club gear hoodies, tees, caps, wraps and shorts with colour toggles built in. Premium feel, part of the same brand experience.

05

Content system

News

A 3D drag carousel for fight results, class updates and announcements. The same carousel format as the Stormie Studios portfolio built from scratch.

06

Social layer

Auto content

Behold.so pulls the gym's Instagram feed live. Post on Instagram the website updates automatically. No backend, no manual work.
